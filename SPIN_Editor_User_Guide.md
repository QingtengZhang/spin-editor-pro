# SPIN 参数编辑器 Pro 操作说明

## 一、程序用途

`spin_editor_pro.py` 是一个用于 `.SPIN` 参数文件的图形化编辑工具，适合批量查看、修改、删除和导出参数。它会尽量保持原文件的行顺序、缩进、等号两侧空格和行内注释；保存或批量写入时默认自动备份原文件。

## 二、启动方式

在程序所在目录打开终端，运行：

```powershell
python spin_editor_pro.py
```

如果系统里有多个 Python，也可以指定 Python 路径运行。

## 三、界面区域

1. 左侧“文件”区域  
   用于打开 `.SPIN` 文件夹、搜索文件、多选文件，以及控制是否包含子文件夹。

2. 中间“参数表”区域  
   显示当前单个 `.SPIN` 文件中的参数。第一列用于勾选批量操作参数，后面依次是参数名、参数值、注释和行号。

3. 右侧“操作”区域  
   提供保存、另存为、差异预览、批量应用、批量删除、批量导出和参数对照表等功能。

## 四、常用流程

### 1. 打开并编辑单个文件

1. 点击“打开文件夹”，选择包含 `.SPIN` 文件的目录。
2. 在左侧文件列表中单击一个文件。
3. 在中间参数表中双击“参数值”直接修改；也可以选中参数后在右侧编辑框修改，再点“应用到表格”。
4. 保存前可点击“预览当前差异”查看修改前后的差异。
5. 点击“保存当前文件”写入磁盘。

### 2. 批量统一修改参数

1. 先打开一个作为模板的 `.SPIN` 文件。
2. 修改需要统一的参数值。
3. 勾选这些参数左侧的“批量”列。
4. 在左侧多选目标文件。
5. 点击“批量应用勾选参数”。

说明：批量应用只会覆盖勾选参数；其他参数会保持目标文件原样。若目标文件缺少某个勾选参数，该参数会追加到文件末尾。

### 3. 批量删除参数

1. 在当前参数表中勾选要删除的参数。
2. 在左侧多选目标文件。
3. 点击“批量删除勾选参数”。
4. 确认后，所选文件中的同名参数行会被删除。

### 4. 批量导出新文件

1. 在参数表中勾选需要统一写入的参数。
2. 在左侧选择需要导出的源文件。
3. 点击“批量导出到文件夹”。
4. 选择目标文件夹。

说明：该功能不会覆盖原文件，会在目标文件夹生成同名 `.SPIN` 文件。

### 5. 查看多个文件的参数差异

1. 在左侧选择多个 `.SPIN` 文件。
2. 点击“生成参数对照表”。
3. 弹出的表格会按“参数名 × 文件名”展示每个文件的参数值。
4. 可点击“导出 CSV”保存对照表。

## 五、搜索和筛选

- 左侧搜索框：按文件名筛选 `.SPIN` 文件。
- 中间搜索框：按参数名、参数值或注释筛选参数。
- “只看勾选”：只显示已经勾选参与批量操作的参数。

## 六、备份规则

默认勾选“自动备份”。

- 保存当前文件时，会在原文件旁生成类似 `filename.SPIN.20260623-153000.bak` 的备份。
- 批量应用或批量删除时，会在当前文件夹下生成 `_spin_backups` 文件夹保存备份。
- 批量导出到新文件夹不会生成备份，因为不会改动原文件。

## 七、快捷键

- `Ctrl+O`：打开文件夹
- `Ctrl+S`：保存当前文件
- `Ctrl+F`：搜索参数
- `Delete`：从表格移除选中的参数行
- `F5`：刷新文件列表

## 八、注意事项

1. “从表格移除”不会立即删除磁盘文件内容，只有点击“保存当前文件”后才会写入。
2. 批量操作只作用于左侧当前选中的文件。
3. 参数名只允许字母、数字、下划线、点和短横线。
4. 如果文件不是 UTF-8，程序会尝试 `gb18030`、`gbk`、`latin-1` 等编码读取，并用读取到的编码写回。
5. 建议首次批量处理前先用少量文件测试一次，并保留“自动备份”。
